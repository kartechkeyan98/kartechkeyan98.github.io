2.1. Span

Definition 2.1 [Linear Combination; Span]

1. [Linear Combination]
Given a finite list of vectors L=(v1​,v2​,⋯vn​), and scalars A=(a1​,a2​,⋯an​)∈Fn, linear combination of L with coefficients in A is:

u=a1​⋅v1​+a2​⋅v2​+⋯+an​⋅vn​≡i=1∑n​ai​⋅vi​​

2. [Finite Spanning Set/Span]
Given a finite list of vectors from V, L=(v1​,v2​,⋯vn​), the finite spanning set or span(L) is the set of all possible linear combinations of L:

span(L)={i=1∑n​ai​⋅vi​:ai​∈F∀i=1,2,⋯n}​

  • If span(L)=V, we say that L spans V.
  • vectors in L are called generators of span(L).

Given V(F)≡ (V,+,⋅,F) is a vector space over the field F≡(F,+^​,⋅^).

Given a finite list of vectors L=(v1​,v2​,⋯vn​), span(L) is the smallest subspace that contains L.

3. [Adding/Removing Independent Vector]

Given a list of vectors in V, call it L=(v1​,v2​,⋯vn​)

1. x∈/span(L) iff span(L)⊊ span(L∪{x})
2. Let x∈L, x∈/span(L−{x}) iff span(L−{x})⊊ span(L)

Proof

This property is a combination of two things:

  1. Contrapositive of P-2.1.2.
  2. Monotonicity of span, which is P-2.1.4.

A⊂B⟹span(A)⊂span(B)​

​A=(u1​,u2​,⋯un​)B=(u1​,u2​,⋯un​,x1​,x2​,⋯xm​)​

These properties are not random. They have an order in which they can appear.

1. [Synthesis]
In the subset sum definition, the simple question was that if given multiple subspaces, is a bigger vector space synthesizable such that it contains the the smaller ones.
The question here is simpler, given a list of vectors, what is the way to create a subspace containing all of them. The answer is span and the bonus is that it is the smallest containing vector space. This is covered in:

  • P-2.1.1 [Smallest Containing Subspace]

2. [Alteration]
View span as an operator. A function that takes in a list of vectors of arbitrary size and produces the smallest vector space containing them all. How will altering the list by one element alter the span, is the question.

  • P-2.1.2 [Adding/Removing Dependent Vectors]
  • P-2.1.2 [Adding/Removing Independent Vectors]

These two properties basically tell when the span is preserved, leading to the concept of dependent and independent vectors.
Span is preserved iff dependent vectors are added or removed while doing the same with indepedent vectors for sure alter the span, what relation the altered span will have comes about from P-2.1.4, which states that span is monotonic.

3. [Combination]
This is one level above alteration. Instead of altering list one vector at a time, the two ways in which two lists can be combined or have a relation is:

  1. Subset: P-2.1.4 [Monotonicity of Span]
  2. Union: P-2.1.5.1 [Span of Union]
  3. Intersection: P-2.1.5.2 [Span of Intersection]

These properties deal with that.

2.2 Linear Independence [LI]

Definition 2.1 [Linear Independence (LI)]

Given a list of vectors L=(v1​,⋯vn​) from V, L is Linearly Independent (LI) if:

∃!{ai​∈F}i=1n​:0V​=i=1∑n​ai​⋅vi​

Meaning, 0V​ is uniquely expressible as a lienar combination of the vectors in the list L.

Some common sense out of this:

  • If L is not LI, it is Linearly Dependent (LD)
  • The empty list ϕ=() is LI by default and span(ϕ)={0V​}.
  • In LI list, the unique linear combination is all zeros.
  • In LD list, a non-zero combination exists which add up to 0V​.
  • The all zeros combination is called trivial linear combination.

Properties of Linear Independence [P-2.2]

Given V(F)≡ (V,+,⋅,F) is a vector space over the field F≡(F,+^​,⋅^).

1. [Uniqueness of Coefficients]

Let L be a finite list of vectors from V,
L is LI ⟺ every vector in span(L) has unique coefficients of linear combination.

NOTE: This uniqueness implies a bijection between Fn & span(L).

Proof

(⟹) Let L be LI. Then, 0V​ is uniquely expressible as only the trivial linear combination, by Def. 2.2.

Take any x∈span(L). Suppose it can be expressed in two ways:

∴∴∴∴∴∴∴​x=i=1∑n​ai​⋅vi​⟶Def. 2.1.2, span=i=1∑n​bi​⋅vi​x+(−x)=i=1∑n​ai​⋅vi​+−i=1∑n​bi​⋅vi​0V​=i=1∑n​ai​⋅vi​+−1F​⋅i=1∑n​bi​⋅vi​→Def. 1.2.5, add. inv.P-1.2.5, mult. with −1F​0V​=i=1∑n​ai​⋅vi​+−1F​⋅(bi​⋅vi​)→Def. 1.2.6.1, Distribitivity &Def. 1.2.2.1, commutativity of + repeated0V​=i=1∑n​(ai​+^​(−1F​⋅^bi​))⋅vi​→Def. 1.2.3.2, Associativity of ⋅& Def. 1.2.6.2, distributivity0V​=i=1∑n​(ai​+^​(−bi​))⋅vi​→P-1.1.7, mult. with −1ai​+^​(−bi​)=0F​∀i=1,2,⋯n⟶Def. 2.2, LIai​=−(−bi​)=bi​∀i=1,2,⋯n⟶Def. 1.1.5.1, add inv.P-1.1.3.1, inverse of inverse ​

This means that any set of coefficients found to express x as linear combination of L is the set of coefficients and is unique.

(⟸) If every vector in span(L) has unique coefficients, since span is a subspace, by P-2.1.1, it should contain 0V​, Def. 1.2.4.1, additive identity.
Then, by what is given, 0V​ has is uniquely expressible, meaning L is LI from Def. 2.2.

Hence, generator is LI iff coefficients in span are unique. q.e.d

2. [Array Representation of Vectors]

Given any finite list L which is LI, has size n.

1. Any vector in span(L) can be represented by a unique array of numbers in Fn.
2. Operations + and ⋅ in span(L) are equivalent to :

  • ⊕:Fn×Fn ↦Fn
    (c1​,c2​,⋯cn​)⊕(d1​,d2​,⋯dn​)
    =(c1​+^​d1​,⋯cn​+^​dn​)

  • ⊙:F×Fn ↦Fn and ⊙:Fn ×F↦Fn
    c⊙(x1​,x2​,⋯xn​)= (x1​,x2​,⋯xn​)⊙c
    =(c⋅^x1​,⋯c⋅^xn​)

3. Fn(F) ≡(Fn,⊕,⊙,F) is a bonafide vector space.
4. span(L)≅Fn, i.e both are isomorphic to each other.

Proof

1. By P-2.2.1, since coefficients of span(L) is unique, and there are n coefficients in total. Hence, each vector in span(L) can be represented by a unique n-tuple from Fn.

2. Given x,y∈span(L), by P-2.2.2.1, we can find xF​,yF​ ∈Fn, which are their equivalent representations s.t.

x=∑i=1n​xF,i​⋅vi​y=∑i=1n​yF,i​⋅vi​​}P-2.2.2.1

x+y can be written as:

∴​x+y=i=1∑n​xF,i​⋅vi​+i=1∑n​yF,i​⋅vi​x+y=i=1∑n​(xF,i​+^​yF,i​)⋅vi​→Def. 1.2.2.1, commutativity of +& Def. 1.2.6.2, Distributivity​

Hence, the equivalent representation of x+y is : xF​⊕yF​ from the above.

c⋅x for any c∈F can be written as :

c⋅x​=c⋅i=1∑n​xF,i​⋅vi​=i=1∑n​(c⋅^xF,i​)⋅vi​→Def. 1.2.6.1, Distributivity &Def. 1.2.3.2, Associativity of ⋅​

Hence, the equivalent representation of c⋅x is : c⊙xF​ by the above.
∴⊕,⊙ are the equivalent of vector add. & scalar mult. in Fn space.

3. This part was actually mentioned in one of the examples below Definition-1.2, that Fn(F) is a bonafide vector space under the elmentwise add. and mult. operations.

4. To establish an isomorphism between span(L) and Fn, two conditions must be satisfied:

  1. Bijectivity: A one-to-one and onto mapping must exist between the two spaces.
  2. Linearity: The mapping must preserve vector addition and scalar multiplication, which was established in P-2.2.2.2.

By P-2.2.1, because L is linearly independent, every vector x∈span(L) can be uniquely represented by a coordinate vector xF​∈Fn. This uniqueness guarantees that the coordinate mapping ϕ:span(L)→Fn is injective (span(L)↪Fn).

Conversely, every coefficient vector xF​∈Fn maps to a vector in span(L) via the linear combination:

x=i=1∑n​xF,i​vi​

Because L is linearly independent, distinct coefficient vectors cannot map to the same vector in span(L). This establishes an injection in the opposite direction (Fn↪span(L)). Since the mapping is both well-defined for all inputs and injective, it constitutes a bijection.

Hence, isomorphism between the two vector spaces is established, span(L)≅Fn

q.e.d

2. Given L is a finite LD list, any finite superset, T⊃L is also LD.
